😄 Are You Mental? – Fun Interactive Web App
📌 Overview

This is a fun and interactive web project built using HTML, CSS, and JavaScript. The application presents a playful question "Are you mental?" with two options: Yes and No. The twist is that the "No" button avoids being clicked, creating an engaging and humorous user experience.

🎯 Features
Clean black + neon themed UI
Elegant and modern typography
Interactive button behavior:
✅ "Yes" button is clickable and displays: "Yes, I knew it"
🚫 "No" button dynamically moves away from the cursor
Smooth animations and transitions for a polished feel
Responsive design for different screen sizes
"No" button always stays within the visible screen area
🛠️ Technologies Used
HTML5
CSS3 (Flexbox, transitions, neon styling)
JavaScript (DOM manipulation, event handling)
🚀 How It Works
The "No" button listens to cursor events and relocates to a random position within the viewport whenever the user tries to approach it.
Smooth transitions are applied to make the movement feel natural.
The "Yes" button triggers a simple message display upon click.
